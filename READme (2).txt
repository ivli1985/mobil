Техническое задание
1. Мобильное приложение для показа фильмов
Цель разработки: создать приложение для фильмов с возможностью их просмотра

2. Функциональные требования
Регистрация и авторизация: возможность регистрации через электронную почту и социальные сети, восстановление пароля 
Профиль пользователя: возможность редактирования личной информации, возможность добавить отзыв к фильму
Главный экран: недавно вышедшие фильмы, популярные на данный момент и комментарии, которые оставил пользователь к фильмам
